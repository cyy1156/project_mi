# Step 0：中间结构 `ContinuousEEG`

## 目标

定义所有数据集共用的中间表示。后面每一步都吃这个结构（或由其衍生的数组），**不再关心**原始是 `.mat` / `.gdf` / `.csv`。

## 你要写的内容

文件建议：`preprocess_lab/src/types.py`

```python
from dataclasses import dataclass
import numpy as np


@dataclass
class ContinuousEEG:
    """连续原始信号（尚未滤波、尚未切 epoch）。"""
    subject: str
    session: str
    x: np.ndarray          # (n_times, n_channels)，单位建议 μV
    fs: float              # 原始采样率
    ch_names: list[str]    # 与 x 的列一一对应
    events: np.ndarray     # (n_events, 2) → [sample_index, event_id]
    labels: np.ndarray | None = None       # 若事件自带类别可同步给出
    artifacts: np.ndarray | None = None    # 试次级伪迹，1=坏
```

辅助检查：

```python
def sanity_check(eeg: ContinuousEEG) -> None:
    assert eeg.x.ndim == 2, f"x 应为 2D，实际 {eeg.x.ndim}D"
    assert eeg.x.shape[1] == len(eeg.ch_names), "通道数与 ch_names 不一致"
    assert eeg.fs > 0, "fs 必须 > 0"
    assert eeg.events.ndim == 2 and eeg.events.shape[1] == 2, "events 应为 (n, 2)"
    if eeg.labels is not None:
        assert len(eeg.labels) == len(eeg.events) or True  # 有的设计 labels 与 events 对齐，按你约定断言
    print(
        eeg.subject,
        eeg.session,
        "x=", eeg.x.shape,
        "fs=", eeg.fs, "Hz",
        "events=", len(eeg.events),
    )
```

## 形状记忆

| 字段 | 形状 / 类型 | 含义 |
|------|-------------|------|
| `x` | `(T, C)` | 时间 × 通道（**不是** `(C, T)`） |
| `events` | `(E, 2)` | `[采样点索引, 事件码]` |
| `artifacts` | `(E,)` 或与 trial 对齐 | `1` 表示坏试次 |

## 常见坑

- MNE 的 `raw.get_data()` 是 `(n_ch, n_times)`，入库前要 **转置成 `(T, C)`**。
- `ch_names` 顺序必须与 `x` 的列一致，后面通道筛选靠名字映射。
