  `d:\cyy\MI\collect_data\LSL_connect_model\LSL_connect_model\.venv`

## 2. 日常实验：启动诱导系统

### 方式 A（推荐 · 操作台 + 真机/合成采数）

1. 打开文件夹：`d:\cyy\MI\experiment_game\`  
2. 双击 **`open_operator.bat`**  
3. 浏览器打开 `http://127.0.0.1:8080/operator.html#setup`（操作台 Setup）  
4. 填写设置后点 **「开始实验」** → **自动打开**诱导页；操作台进入 Run  

**真机 SOP：**

1. 关闭 OpenBCI GUI 的串口/Serial 直播（勿与 BrainFlow 抢 COM）  
2. Cyton USB 已连接，设备管理器确认 COM 号  
3. Setup：开启采集 → Cyton → 串口填实际 COM → 正式 trial 建议首测 4～8  
4. 可选：勾选跳过适应/学习以缩短首测  
5. 开始实验；诱导页「已连接」后按提示走流程；准入按 **G**  
6. 结束后在 Summary 打开会话文件夹，确认有 `eeg.csv`、`continuous/`、`alignment/verify_report.json`  

### 方式 B：仅诱导页快捷方式（默认不采 EEG）

1. 双击 **`open_induction.bat`**  
2. 黑窗口不要关；浏览器打开诱导页  
3. 默认**不采 EEG**；正式采数请用方式 A  

### 方式 C：命令行（专家）

```powershell
cd d:\cyy\MI

# 操作台
.\collect_data\LSL_connect_model\LSL_connect_model\.venv\Scripts\python.exe `
  -m experiment_game.tools.open_operator

# 旧：诱导页 + 合成采数
.\collect_data\LSL_connect_model\LSL_connect_model\.venv\Scripts\python.exe `
  -m experiment_game.tools.open_induction --with-acq --acquire-trials 8

# 旧：诱导页 + 真机
.\collect_data\LSL_connect_model\LSL_connect_model\.venv\Scripts\python.exe `
  -m experiment_game.tools.open_induction --with-acq --real --port COM5 --acquire-trials 8
```

### 步骤 2.1 确认页面已连上

**操作台**右上角应显示「已连接」。诱导页左下角应显示「已连接」/「已就绪」。

**诱导页**检查：

| 位置 | 正常时应看到 |
|------|----------------|
| 左下角状态 | 「已连接」或「已就绪，等待流程…」 |
| 中间 | 第一人称双手 + 桌面 + 目标物 |
| 若未连接 | 先确认操作台黑窗口仍在，再 Ctrl+F5 |

### 步骤 2.2 端口被占用时

若 bat 起不来、提示端口占用：

```powershell
# 查看占用
Get-NetTCPConnection -LocalPort 8080,8765 | Select-Object LocalPort, OwningProcess

# 结束占用进程（把 PID 换成上面看到的）
Stop-Process -Id <PID> -Force
```

然后再双击 `open_operator.bat`。

## 3. 实验进行中的完整阶段

会话顺序固定为：

```text
适应（adapt） → 学习（learn Step1→2→3） → 准入（gate） → 正式采集（acquire）
```

每个 trial 大致时间轴（正式时序，约 17 s）：

| 阶段 | 时长 | 被试做什么 |
|------|------|------------|
| Fixation | 2 s | 看十字，放松 |
| Cue | 2 s | 记住左手/右手提示 |
| MI | 4 s | **想象**该手抓取（正式段手不动画） |
| PostMI | 1 s | 保持 |
| Rest | 4 s | 静息，不要想象 |
| Transition | 3 s | 短暂视角过渡（可稍走神，不进训练窗） |

### 3.1 适应（Adapt）

1. 弹出「环境适应」说明  
2. **被试**：空格 / Enter，或点「我明白了」  
3. 观看左右手各一次**完整抓取并取走杯子**的示意（只需看）  
4. 再确认进入学习

### 3.2 学习（Learn）

共 3 步，每步前有弹窗「开始本步」：

| Step | 画面 | 被试 |
|------|------|------|
| 1 | 完整抓取并取走目标物 | 只观察 |
| 2 | 手明显前伸，不抓握 | 边看边同步想象 |
| 3 | 双手静止 | 自主左右手想象 |

每步默认若干 trial（可用 `--learn-trials` 改）。

### 3.3 准入（Gate）——容易卡住，请注意

弹窗标题类似：**「正式采集准入（人工确认）」**

| 谁 | 能否确认 |
|----|----------|
| 被试空格 | 也可确认（已放宽，避免卡死） |
| **操作者** | 推荐：按 **G**，或点「代确认」，或点弹窗大按钮 |

确认后才进入正式采集。

### 3.4 正式采集（Acquire）

- 双手**静态可见**，无抓取动画  
- 按提示做 Left / Right 想象与 Rest  
- **每 2 个 trial 换物品**（杯子 → 瓶子 → 苹果…）  
- **每 10 个 trial 换场景主题**（家庭 / 医院 / 学校桌面色调）  
- 换物/换景时 Transition 幅度更大  

试次数由启动参数决定，例如：

```text
--acquire-trials 40
```

## 4. 按键与按钮说明

### 4.1 被试

| 操作 | 作用 |
|------|------|
| **空格** / **Enter** | 确认当前说明弹窗（准入页也可用来推进） |
| 点弹窗按钮 | 同上 |

被试**不能**用按键直接操控 3D 双手；手只由程序演示或保持静止。

### 4.2 操作者（左上角条 + 键盘）

| 按键 | 按钮 | 作用 |
|------|------|------|
| **P** | 暂停 | 暂停/恢复计时 |
| **N** | 代确认 | 代点当前弹窗 |
| **G** | （键盘） | 准入确认 |
| **R** | Reject | 标记当前 trial 无效（写入 `trial_reject`） |
| **Esc** | 中止 | 紧急结束整场（会二次确认） |

操作者条会显示：`phase`、`trial`、`obj`、`scene`、`reject` 计数等。

## 5. 实验结束后

1. 页面提示「本会话结束」  
2. **先关浏览器标签，再关黑色命令窗口**（或等窗口自己结束）  
3. 找到本次数据目录：

```text
d:\cyy\MI\experiment_game\data\sessions\
  <subject>_<session>_<年月日_时分秒>\
    session.meta.json
    events.jsonl          ← 事件与标签
    eeg.csv               ← 脑电（仅开启采数时才有）
    eeg.csv.meta.json     ← 可选
```

### 5.1 重要：`--no-acq` 时没有 eeg.csv

| 启动方式 | 有无 eeg.csv | 能否做 Phase4 切窗 |
|----------|--------------|--------------------|
| 默认 `open_induction.bat` | 无 | 否（只有流程演练） |
| `--with-acq` / 不加 `--no-acq` 的 phase2 | 有 | 是 |

正式要训练数据，必须用**带采集**的启动方式。

## 6. 离线切窗（Phase 4）

把会话切成训练张量：`(N, 1, 8, 1000)` + `y_task` / `y_three`。

### 步骤 6.1 运行切窗

```powershell
cd d:\cyy\MI

.\collect_data\LSL_connect_model\LSL_connect_model\.venv\Scripts\python.exe `
  -m experiment_game.tools.run_phase4_epochs `
  --session experiment_game\data\sessions\<会话目录名>
```

成功时终端出现：`PHASE4_OK`，并打印 `N=...`、`y_task=...`、`y_three=...`。

### 步骤 6.2 查看输出

```text
d:\cyy\MI\experiment_game\data\epochs\<会话目录名>\
  X.npy              # (N,1,8,1000)
  y_task.npy         # 0=Rest, 1=Task
  y_three.npy        # 0=Rest, 1=Left, 2=Right
  trial_ids.npy
  meta.json
  train_X.npy / train_y_task.npy / ...
  val_X.npy / val_y_task.npy / ...
  split.json
```

### 步骤 6.3 切窗规则（默认）

- 只用 `phase == acquire` 的事件  
- 锚点：`mi_start`（任务）、`rest_start`（静息）各约 4 s  
- **不用** `cue` 作训练窗起点（避免把提示画面切进训练）  
- 被 Reject 的 trial 不进入集合  
- 通道映射到：`C3, C4, Cz, CP3, CP4, CPz, FC3, FC4`  
- 预处理：CAR → 陷波 50 Hz → 带通 8–30 Hz → 基线 → 重采样/标准化  

常用可选参数：

```powershell
# 也切 adapt/learn（一般不用于主训练）
... --phases all

# 自定义输出目录
... --out experiment_game\data\epochs\my_run

# 不写 train/val 分割
... --no-split
```

更细检查项见：[Phase4_流水线检查清单.md](./Phase4_流水线检查清单.md)。

## 7. 对接训练（train_lab）

1. 确认 `data/epochs/<名>/` 下已有 `train_X.npy`、`train_y_task.npy`、`val_X.npy`、`val_y_task.npy`  
2. 将 `train_lab` 的数据目录指向该文件夹（或拷贝到其约定的 `out/...` 路径）  
3. 确认模型配置：`n_chans=8`，`n_times=1000`  
4. 当前主任务头通常用 **`y_task`（Rest vs Task）**；`y_three` 留给三分类扩展  

> synthetic 数据只能验证流水线，**不能**当作真实分类效果依据。

## 9. 常见问题

### Q1：打开网页后什么都不能点？

- 黑窗口是否还在？  
- 左下角是否「已连接」？  
- 是否停在**准入弹窗**？请操作者按 **G** 或点「代确认」/弹窗按钮。  
- Ctrl+F5 后再试。

### Q2：手只会抖、不伸、不取杯？

- 确认已重启服务并强刷（旧前端曾有动画被重置的问题）。  
- 适应/学习 Step1 才会完整取物；正式段故意无动画。

### Q3：切窗报缺少 eeg.csv？

- 该次用了默认「不采 EEG」。请用 `--with-acq` 重采。

### Q4：bat 乱码 / 找不到路径？

- 请用 **`open_induction.bat`**（纯英文脚本）。  
- 在仓库根用第 2 节命令行方式启动。

### Q5：8080 端口占用？

- 见第 2.2 节结束旧进程后再开。
